# 🚗 Fleet Management System with Real-Time Tracking

A web-based application to track multiple vehicles on an interactive map, calculate optimized routes, and visualize movement using animated markers.

---

## Features

- Track multiple vehicles simultaneously on a map
- Calculate optimized routes using GraphHopper API
- Visualize routes using polylines
- Smooth marker animation along calculated paths
- Real-time map interaction (click to set destination)
- Latitude/Longitude input with validation
- Map zooming, scaling, and panning

---

## 🛠 Tech Stack

- React.js
- JavaScript (ES6+, Async/Await)
- Google Maps API
- GraphHopper Routing API
- HTML5 / CSS3
- Bootstrap
